vue中使用qs插件

本文最后更新于：19 天前

之前用Vue+element写了一个后台管理系统，在使用axios请求数据传参时无法正常的获取数据，百度了一些资料，发现原因是传递参数要将参数序列化。于是使用了qs插件，简单来说，qs 是一个增加了一些安全性的查询字符串解析和序列化字符串的库。

安装插件

npm install qs

在main.js中引入qs，并配置全局

import qs as "qs"

配全局属性配置，在任意组件内可以使用this.$qs获取qs对象 
Vue.prototype.$qs = qs

使用

将对象序列化，多个对象之间用&拼接
let params = qs.stringify({
    id: 1, 
    value: "手机"
})

* qs.parse()和qs.stringify()相反，是将序列化的内容解析成对象的形式,
